# TO-K: guía de usuario del ecosistema familiar

TO-K es un pequeño ecosistema de aplicaciones familiares pensado para acercar a las personas a través de gestos sencillos, útiles y cotidianos.

No es una red social, no es una app de productividad y no pretende sustituir la presencia humana. TO-K funciona como una especie de alma electrónica para la familia: una capa discreta que conecta objetos, personas, recuerdos, cuidados y pequeñas rutinas del día a día.

Cada familia tiene su propio hogar digital, llamado TO-K HOME. Desde ese hogar se conectan distintos satélites: perfiles personales, llaveros para niños, placas para mascotas, seguimiento de lectura y otros pequeños puntos de contacto. Cada satélite cumple una función básica, pero emocionalmente valiosa: avisar, recordar, acompañar, compartir o cuidar.

## 1. La idea central

TO-K nace para convertir objetos sencillos en puntos de conexión familiar.

Un llavero, una placa, una tarjeta o un código pueden abrir una experiencia digital pensada para ayudar a una familia en momentos concretos:

- Cuando alguien encuentra una mascota.
- Cuando un niño necesita que contacten con su familia.
- Cuando una persona quiere compartir sus datos de contacto de forma privada.
- Cuando la familia quiere tener a mano la WiFi de casa.
- Cuando se quiere registrar una pequeña rutina de cuidado.
- Cuando se quiere celebrar que alguien ha terminado un libro.
- Cuando un hogar quiere tener un pequeño centro de actividad común.

La filosofía de TO-K es sencilla: tecnología pequeña, humana y útil.

## 2. Qué compone el ecosistema TO-K

El ecosistema está formado por una central y varios satélites.

### TO-K HOME

Es la centralita familiar. Representa el hogar digital de la familia y conecta el resto de satélites.

Desde HOME se puede:

- Activar el sistema familiar.
- Configurar el nombre del hogar.
- Guardar datos básicos compartidos, como la red WiFi familiar.
- Ver alertas y actividad de los satélites.
- Recibir avisos cuando alguien escanea un satélite importante.
- Consultar notas familiares.
- Acceder a los distintos miembros o elementos conectados.
- Personalizar la imagen, el estilo y algunos datos del hogar.

HOME es el punto común. Si TO-K tuviera un corazón, estaría aquí.

### TO-K ONE

Es el satélite personal. Está pensado para una persona concreta.

Puede funcionar como una tarjeta personal inteligente, un perfil de contacto o una forma rápida de compartir información útil.

Con TO-K ONE se puede:

- Mostrar un perfil personal.
- Compartir teléfono, WhatsApp, email o redes sociales.
- Mostrar enlaces importantes.
- Enseñar la WiFi del hogar si la familia lo permite.
- Configurar contactos ICE, es decir, contactos para emergencias.
- Proteger zonas privadas con PIN.
- Usar sesiones temporales para que cierta información no quede siempre abierta.

TO-K ONE sirve para presentarse, conectar y compartir lo justo, sin depender de redes sociales grandes.

### TO-K KIDS

Es el satélite pensado para niños y niñas.

Su función principal es ayudar a que, si alguien encuentra a un menor o necesita contactar con su familia, pueda hacerlo de manera rápida, clara y segura.

Con TO-K KIDS se puede:

- Mostrar un mensaje de ayuda en varios idiomas.
- Permitir llamada o WhatsApp a la familia.
- Enviar un aviso a TO-K HOME cuando el satélite se escanea.
- Compartir ubicación si la persona que ayuda decide enviarla.
- Mostrar datos básicos configurados por la familia.
- Proteger la configuración con PIN familiar.

TO-K KIDS no está pensado para exponer información innecesaria. Está pensado para actuar en momentos simples pero importantes: avisar, orientar y reunir.

### TO-K PETS

Es el satélite para mascotas.

Convierte una placa o identificador en una pequeña ficha de ayuda para que, si alguien encuentra a la mascota, pueda contactar rápidamente con su familia.

Con TO-K PETS se puede:

- Mostrar el nombre de la mascota.
- Mostrar una foto o icono identificativo.
- Permitir llamada o WhatsApp al dueño.
- Enviar mensajes de ayuda en varios idiomas.
- Enviar ubicación cuando alguien encuentra a la mascota.
- Registrar alertas en TO-K HOME.
- Llevar una pequeña bitácora de cuidados.
- Registrar acciones como comida, paseo, medicina, baño o juego.
- Añadir incidencias de salud o notas importantes.

TO-K PETS acerca el cuidado diario y convierte una placa en algo más vivo: una conexión directa entre quien encuentra y quien espera.

### TO-K BOOKS

Es el satélite lector.

Está pensado para familias que quieren acompañar el hábito de lectura de una forma sencilla y emocional.

Con TO-K BOOKS se puede:

- Buscar libros.
- Guardar libros en una lista de deseos.
- Marcar libros como en lectura.
- Registrar progreso.
- Marcar libros como terminados.
- Sumar puntos o pequeños hitos lectores.
- Celebrar avances dentro de la familia.
- Enviar actividad de lectura a TO-K HOME.

TO-K BOOKS no convierte la lectura en una competición. La convierte en una pequeña memoria compartida: lo que se lee, lo que se termina y lo que apetece descubrir.

## 3. Activación del sistema

El ecosistema empieza con TO-K HOME.

La familia recibe un código de activación junto con su pack TO-K. Ese código es la llave inicial para crear el hogar digital.

El proceso de activación sería así:

1. La familia abre TO-K HOME desde el enlace o el QR/NFC incluido.
2. Introduce el código de activación del pack.
3. El sistema valida el código.
4. Se crea el hogar familiar.
5. La familia configura el nombre del hogar.
6. Se pueden añadir datos básicos, como la WiFi de casa o una imagen familiar.
7. HOME queda preparado para conectar los satélites.

Una vez activado, cada satélite se vincula al hogar mediante su identificador. Así, cuando alguien escanea TO-K KIDS, TO-K PETS, TO-K ONE o TO-K BOOKS, la información y las acciones quedan conectadas al mismo núcleo familiar.

## 4. Cómo se usaría en el día a día

TO-K está pensado para no exigir atención constante. No es una aplicación para mirar todo el rato. Es una capa de apoyo.

La familia puede usarlo de formas muy distintas:

- Como centro familiar para ver pequeños avisos.
- Como sistema de emergencia ligera para niños o mascotas.
- Como tarjeta personal de contacto.
- Como memoria de cuidados de una mascota.
- Como registro lector familiar.
- Como forma bonita de conectar objetos físicos con acciones digitales.

La magia está en que cada acción es pequeña. Un escaneo. Un aviso. Una ubicación. Un libro terminado. Una nota de salud. Una llamada rápida. Una WiFi compartida. Cosas básicas, pero con sentido.

## 5. Acciones principales de TO-K HOME

HOME es la pantalla que da contexto a todo.

Acciones familiares dentro de HOME:

- Crear el hogar familiar.
- Personalizar el nombre del hogar.
- Añadir una foto o imagen representativa.
- Configurar la WiFi familiar.
- Mostrar un QR de WiFi para visitas.
- Consultar alertas recientes.
- Ver escaneos de satélites.
- Recibir avisos de KIDS o PETS.
- Consultar actividad de lectura de BOOKS.
- Consultar pequeñas notas familiares.
- Acceder a configuración protegida.

HOME funciona como un tablero tranquilo. No busca saturar. Solo recoge lo importante.

## 6. Acciones principales de TO-K ONE

TO-K ONE está pensado para una persona.

Acciones del satélite ONE:

- Mostrar nombre, foto y una bio corta.
- Compartir teléfono principal.
- Compartir WhatsApp.
- Compartir Telegram, email o web.
- Mostrar redes sociales.
- Crear una tarjeta de contacto rápida.
- Mostrar la WiFi del hogar si está permitida.
- Mostrar contactos de emergencia.
- Activar un modo de sesión temporal.
- Proteger la configuración con PIN.

Ejemplo de uso:

Una persona puede llevar una tarjeta TO-K ONE en la cartera, en un llavero o en un objeto personal. Al escanearla, otra persona ve una forma clara y cuidada de contactar, sin tener que buscar perfiles en varias redes.

## 7. Acciones principales de TO-K KIDS

TO-K KIDS está pensado para situaciones donde importa actuar rápido.

Acciones del satélite KIDS:

- Mostrar un aviso claro de ayuda.
- Mostrar mensajes en español, inglés y alemán.
- Permitir llamada a la familia.
- Permitir WhatsApp a la familia.
- Permitir SMS si está configurado.
- Pedir a la persona que ayuda que envíe ubicación.
- Registrar en HOME que el satélite ha sido escaneado.
- Registrar si alguien ha indicado que está con el niño o niña.
- Proteger configuración con PIN familiar.

Ejemplo de uso:

Si alguien encuentra a un niño o niña con un llavero TO-K KIDS, puede escanearlo y ver instrucciones simples. La familia puede recibir una alerta y, si la persona que ayuda lo permite, una ubicación aproximada.

La idea no es vigilar. La idea es facilitar el reencuentro.

## 8. Acciones principales de TO-K PETS

TO-K PETS ayuda cuando una mascota se pierde, pero también acompaña sus rutinas.

Acciones de emergencia:

- Mostrar nombre de la mascota.
- Mostrar foto, tipo de animal o descripción.
- Permitir llamada al dueño.
- Permitir WhatsApp al dueño.
- Mostrar mensaje de ayuda en varios idiomas.
- Enviar ubicación si quien encuentra acepta compartirla.
- Registrar en HOME que la mascota ha sido escaneada.

Acciones de cuidado diario:

- Registrar comida.
- Registrar paseo.
- Registrar baño.
- Registrar medicina.
- Registrar limpieza.
- Registrar juego.
- Añadir incidencias de salud.
- Consultar bitácora reciente.

Ejemplo de uso:

La placa de una mascota deja de ser solo un nombre y un teléfono. Se convierte en un puente. Si alguien la encuentra, sabe qué hacer. Si la familia la usa en casa, también puede guardar pequeños cuidados del día.

## 9. Acciones principales de TO-K BOOKS

TO-K BOOKS acompaña el hábito lector sin hacerlo pesado.

Para el uso operativo diario y la entrega de este satélite, consultar también `README-OPERATIVO-TOK-BOOKS.md`.

Acciones del satélite BOOKS:

- Buscar libros por título o autor.
- Añadir libros a lista de deseos.
- Marcar libros como en lectura.
- Actualizar progreso.
- Marcar libros como terminados.
- Registrar libros leídos durante el año.
- Ver libros pendientes o terminados.
- Compartir hitos lectores con HOME.
- Sumar puntos simbólicos de lectura.

Ejemplo de uso:

Una familia puede celebrar cuando alguien termina un libro, guardar futuras lecturas o construir una pequeña memoria lectora. No se trata de medirlo todo, sino de dar valor a una costumbre bonita.

## 10. Privacidad y control familiar

TO-K está pensado para que la familia mantenga control sobre la información.

Cada satélite muestra solo los datos que la familia decide configurar. La información sensible debería estar protegida por PIN o por permisos de acceso. Las acciones públicas, como llamar, enviar WhatsApp o compartir ubicación, están pensadas para momentos concretos de ayuda.

Buenas prácticas de uso:

- No poner información innecesaria en perfiles públicos.
- Revisar los teléfonos de contacto con frecuencia.
- Usar un PIN familiar que no sea obvio.
- Mantener actualizada la foto o descripción de mascotas.
- Probar cada satélite después de configurarlo.
- Explicar a niños y familiares mayores para qué sirve cada TO-K.

TO-K funciona mejor cuando la familia lo entiende como una herramienta de apoyo, no como una herramienta de control.

## 11. La experiencia emocional

TO-K intenta que la tecnología sea menos fría.

Cada satélite tiene una función práctica, pero detrás hay una intención emocional:

- HOME da pertenencia.
- ONE da identidad.
- KIDS da tranquilidad.
- PETS da cuidado.
- BOOKS da memoria.

La familia no necesita abrir TO-K todos los días para que tenga valor. Su valor aparece cuando hace falta: cuando alguien necesita contactar, cuando una mascota se pierde, cuando un familiar quiere compartir algo útil, cuando un libro se termina o cuando una pequeña rutina queda registrada.

Por eso TO-K puede entenderse como un alma electrónica: una presencia ligera que acompaña a los objetos y los convierte en pequeñas puertas hacia la familia.

## 12. Recorrido completo de una familia

Ejemplo de uso completo:

1. La familia recibe su pack TO-K.
2. Activa TO-K HOME con el código incluido.
3. Configura el nombre del hogar y la WiFi.
4. Vincula un TO-K ONE para una persona adulta.
5. Vincula un TO-K KIDS para un llavero infantil.
6. Vincula un TO-K PETS para la placa de una mascota.
7. Vincula TO-K BOOKS para acompañar la lectura.
8. Cada satélite queda conectado a HOME.
9. Cuando ocurre una acción importante, HOME registra la actividad.
10. La familia tiene un pequeño ecosistema digital propio.

El resultado no es una app aislada. Es una red familiar de pequeños gestos.

## 15. Resumen breve

TO-K es un ecosistema familiar compuesto por una central, TO-K HOME, y varios satélites: ONE, KIDS, PETS y BOOKS.

HOME organiza el hogar digital. ONE comparte identidad y contacto. KIDS facilita avisos familiares en situaciones de ayuda. PETS conecta mascotas perdidas o cuidados diarios con su familia. BOOKS acompaña la lectura y transforma avances lectores en memoria compartida.

El sistema se activa con un código familiar y cada satélite se vincula al hogar. A partir de ahí, la familia puede usar TO-K como una capa ligera de conexión: avisos, llamadas, ubicación, rutinas, WiFi, lectura y pequeños registros.

TO-K es una forma de acercar la tecnología a lo cotidiano: sencilla, cálida y con alma electrónica.
